# CS-2640-Group1-Final-Project

# Purpose
Use W, A, S, D to move the snake and collect the apples to grow your snake.

# How to play
(1) Open SnakeGame.asm in MARS
(2) Navigate to "Tools" at the top left of MARS
(3) Click "Bitmap Display"
(4) Set Unit Height in Pixels to 8
(5) Set Unit Width in Pixels to 8
(6) Click "Connect to MIPS"
(7) Open "Tools" again
(8) Click "Keyboard and Display MMIO Simulator"
(9) Connect to MIPS
(10) Assemble code
(11) Run and then IMMEDIATELY click on the keyboard box (bottom box) of "Keyboard and Display MMIO Simulator"
(12) Use WASD to play
